配置 FTP

首先安装ftp

如果说出现

说明你电脑已经安装过了或者已经是最新版

最好使用匿名用户登录，不要轻易暴露你的密码

#创建FTP用户组ftp-users
groupadd ftp-users

#创建用户组目录ftp-docs
mkdir /home/ftp-docs

#修改权限
chmod -R 750 /home/ftp-docs

#750组只能读不能写，要组能写，请改成770或760

指定文件夹归属用户和用户组
chown root:ftp-users /home/ftp-docs

#添加FTP用户并为他们设置缺省目录
useradd -g ftp-users -d /home/ftp-docs

#设置密码passwd user1
